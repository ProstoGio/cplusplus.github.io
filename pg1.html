Through an int, a box is created in the computer's memory, with the name whatever you call it. For example: int a; He created a drawer called "a". But you can name not only the letter, but also the name. For example, if we want a cell in which the sum of two numbers will be stored, we can create it and name it like this: int sum; If, for example, the number of five-digit numbers, then so: int fivedigit_quantity; and so on. we can name it what we want.


Now, we must learn how to do mathematical operations on numbers what we input, but first we must to learn something until that. There is something in C++ that is much needed. This is an assignment. We can see this in that program upper: "sum = a + b;". this is assignment. We told our program to assignment "a+b" to sum. that means that if a=1 and b=2, sum will equal to 3. You can also subtract, multiply, divide, and take the remainder. subtract_"-". multiply_"*". divide_"/". take the remainder_"%".
