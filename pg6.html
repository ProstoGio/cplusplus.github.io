Now we will talk about, how logical operators work on string. on string we can use only one logical operator, "+". Other logical operators aren't working on string. Let me explain, how "+" works on string. I will use example:


        string s1, s2, s3;
        s1= "Hello";
        s2= "World";
        s3= s1 + s2;
        cout << s3;

"+" Operator connects any quanity of strings. if you understand, what will be output? output will be: "HelloWorld". programm connected s1 and s2 and got string "HelloWorld". If you want to insert space between s1 and s2, then add one space to end to s1, or one space in start of s2. Adding 1 space at the end of s1:


        string s1, s2, s3;
        s1= "Hello ";
        s2= "World";
        s3= s1 + s2;
        cout << s3;

Adding 1 space at the start of s2:


        string s1, s2, s3;
        s1= "Hello";
        s2= " World";
        s3= s1 + s2;
        cout << s3;

Output in both situations will be: "Hello World".
