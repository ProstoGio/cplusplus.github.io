Arrays



Today we will study a not so difficult topic, but difficult compared to previous topics, a topic called "arrays". An array is not a specific function or operator. Let's get started.


An array is a group of numbers or other things. For example, people saw a forest. They named it "wood". There are 5 trees in total in this forest, and each one has its own number, index. (Note!!! The number of the first element in the array is 0)


        First tree index: "0";
        Second tree index: "1";
        Third tree index: "2";
        Fourth tree index: "3";
        Fifth tree index: "4".

Now we will teach you how to display these elements. If we want to select, for example, the third tree, we write like this: â€œ int a=wood[2]; ". After that, what is in the second element of the wood array will be stored in some variable "a". and so on. We can write whatever we want in the array. For example: let's create an array called "a" with 5 elements.


int a[5];
In this case the array is empty.
        
for ( int i = 0; i < 5; i++ ) a[ i ] = 0;

After that, the array will look like this: [0, 0, 0, 0, 0]
we wrote zeros in it. Let's assign 10 to the fifth element.

a[4]=10;

After that the array will look like this: [0, 0, 0, 0, 10]
Now assign 7 to the second one.

a[1]=7;

After that it will become like this: [0, 7, 0, 0, 10]

Now that we have enough knowledge about arrays, let's do one task as usual. The condition is as follows: we introduce an array with n number of elements and increase each member by one. Let's do this.



        int n;
        cin >> n;
        int a [n];
        for (int i = 0; i < n; i++) {
            cin >> a [i];
            a [i] = a [i] + 1;
        }
        for (int i = 0; i < n; i++) {
            cout << a [i];
        }


this is a code. it fully works. if you see, we assign +1 to each array element. if you input 5 and [1, 2, 3, 4, 5] it will output [2, 3, 4, 5, 6]. thats all.
